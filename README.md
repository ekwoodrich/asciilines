## Functionality
Accepts a tvg file as a command line argument and renders it, outputing the rendered canvas to stdout.

## How to run
python3 asciilines.py input.tvg

### Example tvg file
``` 
3 4
* 1 -1 h 5
# -1 1 v 5 
```
